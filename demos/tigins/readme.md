# Tigins
Tig installer

Steps

1. Check if tig is installed 
    - If yes, show version and exit
    - If no, download pre compiled binary from github releases
2. Save binary to ~/.tig/bin/ on unix systems or %APPDATA%\tig\bin\ on windows
3. Download stdlib
4. Save stdlib to ~/.tig/lib/ on unix systems or %APPDATA%\tig\lib\ on windows
