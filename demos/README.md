# Demos

This section of the repo has simple demos of the language being put to use in practical scenarios.
They have different levels of complexity and are meant to showcase the language's capabilities.

All projects have to be written in Tig.

Tig's ecosystem philosophy is Unix like, we dont have one tool that does everything but rather a suite of tools each handcrafted for one job that has to be done right.

## Highlighted projects

Choix - A linter for Tig, checks for memory safety and anti-patterns.
Tiger - Build system for Tig. 
Fang - Package manager for Tig. 
tdb - A debugger for Tig. 

bf - A brainfuck interpreter written in Tig
