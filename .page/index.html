i32 fn fibonacci: i32 n {
    if (n <= 1) {
        ret n
    }
    ret fibonacci(n - 1) + fibonacci(n - 2)
}

hot i32 fn main: =>u8 args {
    i32 result = fibonacci(10)
    printf("Fibonacci(10) = %d\n", result)
    ret 0
}

Why Tight-C?

🔥

Hot Reload

Mark functions with hot and update code at runtime without restarting your application.

⚡

Zero Overhead

Compiles directly to C with minimal abstractions. What you write is what you get.

🔒

Memory Safety

Fat pointers, slices, and compile-time checks help prevent common memory bugs.

🔄

C Interop

Use any C library with extern "C" blocks. No FFI overhead.

📦

Simple Syntax

Clean, modern syntax without sacrificing performance. No preprocessor needed.

🎯

Compile-Time Checks

Scope checking, type safety, and pinning protection built into the compiler.

Quick Start

# Install from source
$ git clone https://github.com/tightc/tcc
$ cd tcc && make

# Compile your program to C
$ ./tcc main.tc -o main.c

# Or compile directly to binary
$ ./tcc main.tc -c myapp

# Enable hot reload
$ ./tcc game.tc -H hot.so -c game

# Get help on any error
$ ./tcc --error E014
