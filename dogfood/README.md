# Demos

This section of the repo contains project ideas for the language to be put to use in practical scenarios.
They have different levels of complexity and are meant to showcase the language's capabilities.

All projects have to be written in Tig.

Tig's ecosystem philosophy is Unix like, we dont have one tool that does everything but rather a suite of tools each handcrafted for one job that has to be done right.

## Projects ideas

Lintig - A linter for Tig, checks for memory safety and anti-patterns.
Fang - Package manager for Tig. 
Tdbug - A debugger for Tig. 
Meic - Make but remade in Tig and with nicer syntax
TIDE - A DOS/Free Pascal 90's style TUI IDE just for Tig and with AI coding support.
